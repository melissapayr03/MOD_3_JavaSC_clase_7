# Dynamic Password Generator
An interactive and responsive interface for generating secure, random passwords. Built with JavaScript and Tailwind CSS to practice DOM manipulation and interactive UI development.

# Technologies
- **Languages:** HTML5, CSS3, JavaScript
- **Frameworks:** Tailwind CSS
- **Version control:** Git & GitHub

# Features
- **Customizable Generation:** Options to include uppercase/lowercase letters, numbers, and special symbols.
- **Dynamic Interactivity:** Real-time DOM manipulation provides instant feedback and interactions.
- **Responsive Design:** Fully optimized for both desktop and mobile devices.
